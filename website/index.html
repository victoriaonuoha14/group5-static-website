Group 5 Cloud Project

Deploying a Static Website on Azure with Full Automation

Welcome to the Future of Cloud

This project demonstrates a fully automated deployment of a unique static website on Microsoft Azure using Infrastructure-as-Code.

Key Features

  • Azure CLI for resource provisioning (no portal!)
  • Bash scripting for end-to-end automation
  • NGINX web server on Ubuntu 22.04
  • GitHub Actions for CI/CD (Bonus!)
  • Accessible via public IP on port 80
